# Data Visualization2

# Seaborn

  • KDE

    • Kernel density estimation, 히스토그램을 스무딩한 것
    • 통계 용어 파트를 살펴 볼 것
    • sns.kdeplot(tips["total_bill"]) → kde plot만 따로 그리기

  • sns.barplot(x ="sex" , y = "total_bill" , data = tips) → sex별로 total bill 막대 그래프 그리기, 기본적으로는 평균으로 그래프가 그려진다

    • estimator = np.std → std(표준 편차)로 내가 원하는 값들을 비교해 줌. 내가 원하는 함수를 넣을 수 있음

  • sns.countplot(x = "sex", data = tips) → 갯수를 bar plot으로 보여줌

  • sns.boxplot(x = "day", y = "total_bill", data = tips) → box plot으로 보여줌

    • hue = "smoker" 을 추가하면, smoker 별로 분리해서 box plot 보여줌

  • sns.violinplot(x = "day", y = "total_bill", data = tips) → violin plot으로 보여줌

    • hue를 사용하면 두개로 나눠서 따로 그려주고, split = True를 사용하면 한쪽 한쪽 다르게 그려줌 (하나의 바이올린 플랏에)
    • 다 적용한 모슴

  • sns.stripplot( x = "day", y = "total_bill", data = tips) → strip plot으로 그려줌

    • jitter = True 를 사용하면 좀 더 넓게 표현하여 뭉쳐져 있는 점들을(밀도 차이를) 더 명확히 볼 수 있음

  • sns.swarmplot( x = "day", y = "total_bill", data = tips )

    • 모든 categorical plot들을 합친것

    • 바이올린 플랏과 같이 그리면 데이터 분석에 좋다

  • sns.factorplot( x = "day", y = "total_bill", data = tips )

    • 그냥 일반적인 플랏
    • kind = barplot, violinplot 등으로 다른 것으로 변경 가능
